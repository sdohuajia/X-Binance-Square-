# X → Binance Square Auto Sync

## 安装
1. 解压 ZIP 到固定文件夹。
2. Chrome 打开 `chrome://extensions/`；Edge 打开 `edge://extensions/`。
3. 开启“开发者模式”，点击“加载已解压的扩展程序”，选择解压后的文件夹。
4. 扩展会自动打开设置页。填写你自己的**币安广场 OpenAPI Key**并保存；保存后输入框会自动清空并隐藏密钥，只显示“已配置”。
5. 回到 `x.com` 刷新一次。以后在此浏览器发新原创推文，会自动同步到币安广场。

## 数据与安全
- OpenAPI Key 只保存在浏览器扩展的 `chrome.storage.local`，不会同步到浏览器云端，也不会发送到开发者服务器；保存后不会重新显示在设置页 DOM 中。
- 扩展不申请剪贴板权限，不读取或复制剪贴板、键盘输入、X 密码、Cookie、私信、浏览历史或下载内容。
- X 页面代码只解析成功的 `CreateTweet` 发推响应，与发推无关的网络请求体会直接跳过。
- 扩展直接调用币安广场官方发布接口。
- 不要将包含你已填写密钥的浏览器用户数据目录分享给他人。
- 卸载扩展会删除扩展本地配置；也可以在设置页点击“删除密钥”。

## 行为
- 首次捕获原创推文时自动绑定当前 X 用户名，避免同一浏览器切换账号后误发到同一个币安广场账号。
- 设置页可以重置 X 账号绑定。
- 原创推文和引用推文会同步文字；回复、转推不自动同步。
- 当前只同步文字，不同步图片和视频；图片附带的末尾 `t.co` 媒体链接会自动移除。
- 发布失败会排队重试；点击扩展图标可看状态。

## 注意
这是无需付费 X API 的浏览器内方案，只能捕获从安装了扩展的桌面浏览器发布的推文；从手机 App 或其他浏览器发布的推文不会被捕获。X 内部接口变更时，扩展可能需要升级。
